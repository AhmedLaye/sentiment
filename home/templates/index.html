Project Description

Le projet Sentiment Analysis est une application web permettant d'analyser les commentaires textuels en termes de sentiment. Les utilisateurs peuvent saisir un commentaire, et notre modèle d'analyse de sentiment déterminera s'il est positif ou négatif. Caractéristiques principales :

  • 📊 Analyse de sentiment en temps réel.
  • 📝 Classification des commentaires en "Positif" ou "Négatif".
  • 🌟 Interface utilisateur conviviale avec des émojis pour indiquer le sentiment.
Public cible :
  • 👥 Les utilisateurs qui souhaitent évaluer rapidement le sentiment d'un commentaire.
  • 👨‍💻 Les développeurs intéressés par l'intégration de l'analyse de sentiment dans leurs applications.
Technologies utilisées :
  • 🐍 Django pour le backend.
  • 🌐 HTML/CSS pour le frontend avec Bulma CSS.
  • 📊 NLTK et Scikit-Learn pour l'analyse de sentiment.
